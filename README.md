# Playwright Automation Framework (Page Object Model)

End-to-end UI test automation framework built with **Playwright** and **Python (pytest)**, 
following the **Page Object Model (POM)** design pattern. Integrated with **GitHub Actions** for automated CI test execution.

## 🚀 Tech Stack
- **Playwright** – Browser automation
- **Pytest** – Test runner & assertions
- **Python** – Core language
- **Docker** – Containerized test execution
- **GitHub Actions** – CI/CD pipeline
